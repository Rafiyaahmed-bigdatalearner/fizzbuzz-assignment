FizzBuzz Assignment

This project is a simple implementation of the classic FizzBuzz problem.

Problem Statement

Print numbers from 1 to 100 with rules:

3 → Fizz
5 → Buzz
3 and 5 → FizzBuzz
otherwise print number


How to Run
Make sure Python is installed.


Run:
python app/main.py


Expected Output

1
2
Fizz
4
Buzz
Fizz
7
8
Fizz
Buzz
FizzBuzz
...
